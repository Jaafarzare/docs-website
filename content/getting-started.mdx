# شروع به کار با پروژه نکست‌جی‌اس (Next.js)

## مقدمه

در این راهنما، شما یاد خواهید گرفت که چگونه یک پروژه جدید با استفاده از نکست‌جی‌اس (Next.js) ایجاد کنید و اولین صفحه خود را بسازید. نکست‌جی‌اس یک فریم‌ورک قدرتمند برای ساخت برنامه‌های وب مدرن با React است که امکاناتی مانند رندرینگ سمت سرور (SSR)، تولید استاتیک (Static Generation) و مسیریابی پیشرفته را ارائه می‌دهد.

## پیش‌نیازها

قبل از شروع، مطمئن شوید که موارد زیر را نصب کرده‌اید:

- [Node.js](https://nodejs.org/) (ورژن 12 یا بالاتر)
- [npm](https://www.npmjs.com/) یا [yarn](https://yarnpkg.com/)

## ایجاد یک پروژه جدید

برای ایجاد یک پروژه جدید با نکست‌جی‌اس، دستور زیر را در ترمینال اجرا کنید:

```bash
npx create-next-app@latest نام-پروژه-شما
```
