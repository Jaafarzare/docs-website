# راهنمای نصب پروژه نکست‌جی‌اس (Next.js)

در این بخش، مراحل نصب و راه‌اندازی پروژه نکست‌جی‌اس (Next.js) را به طور کامل توضیح خواهیم داد. با دنبال کردن این مراحل، می‌توانید پروژه خود را به سرعت راه‌اندازی کنید.

## پیش‌نیازها

قبل از شروع، مطمئن شوید که موارد زیر را نصب کرده‌اید:

- **[Node.js](https://nodejs.org/)** (ورژن 12 یا بالاتر)
- **[npm](https://www.npmjs.com/)** یا **[yarn](https://yarnpkg.com/)** (برای مدیریت پکیج‌ها)

برای بررسی نسخه‌های نصب‌شده، می‌توانید دستورات زیر را در ترمینال اجرا کنید:

```bash
node -v
npm -v
```
