# مرجع API پروژه نکست‌جی‌اس (Next.js)

در این بخش، مرجع کامل API برای پروژه نکست‌جی‌اس (Next.js) ارائه شده است. این مرجع شامل توابع، کامپوننت‌ها و قابلیت‌های اصلی است که می‌توانید در پروژه خود استفاده کنید.

## توابع اصلی

### `getStaticProps`

این تابع برای تولید صفحات استاتیک استفاده می‌شود. داده‌های مورد نیاز صفحه در زمان ساخت (build time) دریافت می‌شوند.

```javascript
export async function getStaticProps(context) {
  return {
    props: {}, // داده‌های مورد نیاز صفحه
  };
}
```
